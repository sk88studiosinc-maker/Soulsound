# Contributing to SoulSound

Thank you for your interest in contributing to SoulSound.

SoulSound is being built **in public** with an intentional, human-centered approach.  
This project values clarity, emotional integrity, and thoughtful design over speed.

## How to Contribute

At this stage, contributions are welcome in the form of:
- Feedback and discussion
- Design or UX suggestions
- Code improvements aligned with the mission
- Documentation enhancements

Before submitting a pull request:
1. Please open an Issue to discuss the idea.
2. Describe *why* the contribution aligns with SoulSound’s purpose.
3. Keep changes focused and respectful of the project’s tone.

## Project Philosophy

SoulSound is not optimized for:
- Growth hacks
- Engagement manipulation
- Extractive design patterns

It *is* optimized for:
- Creativity
- Emotional presence
- Gentle, intentional technology

## Code of Conduct

Be kind.  
Assume good intent.  
Build as if people’s inner lives matter.

---

If you’re unsure whether something fits, opening a conversation is always welcome.
